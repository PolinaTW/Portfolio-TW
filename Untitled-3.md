# **Инструкция** 
Данное СПО XY (условное название) используется для управления сервисами (службами), интегрированными с ним.  
**Список сервисов, которыми может управлять СПО XY**:   
- 3gppd  
- Dbservice   
- JMservice       

# **Порядок добавления сервиса**  
**ШАГ 1**  
Запустить СПО — при первом запуске откроется окно программы без добавленных сервисов - Скриншот 1:    
![]()  
<p style="text-align: center;">Скриншот 1</p>  

**ШАГ 2**  
На верхней панели нажать кнопку **«+»** для добавления сервисов в программу.  

**ШАГ 3**    
В открывшемся окне “Добавление нового сервиса” ввести наименование сервиса из выпадающего списка (Скриншот 2).  
![]()  
<p style="text-align: center;">Скриншот 2</p>    

**ШАГ 4**    
Настроить режимы сервиса:  
- **Автоматически перезапускать при остановке**. В данном режиме сервис автоматически перезапускается при отсутствии ответа в течение 30 секунд.  
- Сохранять log-файлы при сбое программы (log’и — журналы событий программы)  
- Сохранять dump-файл при сбое программы (dump’ы — снимки оперативной памяти данных/копия базы данных).    

**ШАГ 5**  
После ввода данных нажать кнопку **Добавить** — сервис добавится к управлению программой XY и уже будет запущен — см. Скриншот 3:   
![]()  
<p style="text-align: center;">Скриншот 3</p>    

# **Порядок управления сервисом**  
В главном окне для каждого сервиса отображается панель управления сервисом: основные настройки и кнопки запуска — Скриншот 4:  
![]()  
<p style="text-align: center;">Скриншот 4</p>   
**Основные настройки сервиса**:  
- Кнопка добавления сервиса **плюс**  
- Кнопка удаления сервиса **корзина** из управления программой  
- Кнопка настроек **шестеренка**: при нажатии откроется окно настроек сервиса, а также информацию о его версии и адресе настроек — Скриншот 5:  
- Также кнопка справа **i** открывает аналогичную информацию о сервисе.  
![]()
<p style="text-align: center;">Скриншот 5</p>    
Подробные сведения об адресе настроек открывается через кнопку “Открыть” (а также через кнопку **“i”**) — в окне “Информация” указаны данные (Скриншот 6):  
- Рабочая директория (адрес папки)
- Директория log-файлов сервиса: адрес журнала событий  
- Файл настроек сервиса: доступен только для просмотра.  

![]()  
<p style="text-align: center;">Скриншот 6</p>      
**Кнопки запуска**:   
- **стрелка** —  запустить остановленный сервис  
- **пауза** — остановить запущенный сервис  
- **круглая стрелка** — перезапустить сервис.
